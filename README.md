# API Monitoring System

Real-time API health monitoring with WebSocket updates.

## Live Demo
- Frontend: https://api-monitoring-front.vercel.app
- Backend: https://api-monitoring-back.onrender.com

## Features
- JWT Authentication
- Real-time monitoring (30s intervals)
- WebSocket live updates
- Uptime statistics
- Response time tracking

## Tech Stack
**Backend:** NestJS, Prisma, PostgreSQL, WebSocket
**Frontend:** Next.js, TypeScript, Tailwind CSS
**Deploy:** Render, Vercel, Neon DB
